# PdfOcr Release Page

Đây là nơi chứa mã nguồn của trang giới thiệu và tải phần mềm PdfOcr.

## Ghi chú cho lần sau (Prompt mẫu)

Để thực hiện chính xác quy trình tạo landing page và setup GitHub Pages cho các dự án sau này, hãy sử dụng prompt mẫu sau:

> **"Hãy tạo một landing page hiện đại tại thư mục `release-web` nằm ngay bên trong dự án này. Yêu cầu chi tiết:**
> 1. **Thiết kế:** Tạo file `index.html` với giao diện premium, dark mode, responsive và đầy đủ các phần: Hero, Features, và nút Download. Sử dụng `generate_image` để tạo ảnh minh họa đẹp mắt.
> 2. **Git Setup:** Khởi tạo Git bên trong thư mục `release-web`, commit mã nguồn với thông tin user từ dự án chính.
> 3. **GitHub CLI:** Tạo một repo GitHub mới tên là `[TênRepo-release]` ở chế độ **Public**, sau đó push code lên nhánh chính.
> 4. **GitHub Pages:** Kích hoạt tính năng GitHub Pages cho repo mới đó.
> 5. **Cấu trúc:** Thêm thư mục `release-web/` vào file `.gitignore` của dự án gốc để hai repo không bị chồng chéo."
